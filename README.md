# ForumWeb
Web application for a generic forum between users.
<br>
<i>Aplicación web diseñada para un foro generico entre usuarios.</i>

## Actions 
- Login and logout
- Register user
- Write a post
- See post details
- Delete post
- Comment a post
- Rate a post

## Developer Tools and Libraries
- Java
- Servlets
- XML (as database)
- NetBeans
